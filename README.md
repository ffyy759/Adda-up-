## 🎯 Kya Hai Adda UP?

> **Adda UP** ek social + gaming Android app hai — **sirf UP walon ke liye!**
> Yahan tum battles karo, territory capture karo, dosto se baat karo, aur games khelo.
> Sab kuch ek jagah — aur sab kuch **Made in UP** 🚀

---

## ⚔️ Features

### 🗳️ Battle Mode
| Battle Type | Description |
|-------------|-------------|
| **Vote Battle** 🗳️ | Do options daalo — log vote karein, jis side zyada votes uski jeet |
| **Prediction Battle** 🔮 | True ya False — sahi predict karo toh coins jito |
| **Roast Battle** 🔥 | Sabse tez roast karo — jeet ka badge aur coins milenge |

### 🗺️ Zone Wars — Territory Capture
- Apne area ki territory capture karo
- Doosron ki zone pe attack karo
- UP ka map control karo

### 💬 Full Chat System
- 📩 **Direct Messages** — private baat karo
- 👥 **Group Chat** — gang ke saath
- 📸 **Stories** — 24 ghante mein gayab

### 🎮 Games
| Game | |
|------|-|
| ♟️ Chess | Classic strategy |
| 🎲 Ludo | Desi classic |
| 🐍 Snake | Old school fun |
| 🔢 2048 | Brain game |

### 🪙 Coins & Rank System

| Task | Coins |
|------|-------|
| 📅 Daily Login | +2 |
| ⚔️ Battle Jeeto | +15 |
| 🗳️ Vote Karo | +1 |
| 🗺️ Zone Capture | +10 |
| 🔮 Sahi Predict | +20 |
| 🤝 Referral | +25 |

**Rank Ladder:**
```
🌱 Navagat → ⚡ Khiladi → 😤 Asuya → 👹 Asur
→ 🦁 Sher → 🏹 Baahubali → 👑 Nawab → 🔱 Adda King
```

## 🚀 Setup Karo

```bash
# Clone karo
git clone https://github.com/ffyy759/Adda-up-.git
cd Adda-up-

# Dependencies install karo
npm install

# Start karo
npx expo start
```

> **Note:** Firebase config apna daalna hoga `app/firebase/config.js` mein
